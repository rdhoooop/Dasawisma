Kebun Dasawisma - Mobile-first Bootstrap template
Files:
- index.html         : Main page
- css/styles.css     : Custom styles (loads after Bootstrap)
- js/main.js         : Client-side interactions (search, share, print placeholder)

How to use:
1. Place the files on a static host (GitHub Pages / Netlify).
2. Edit index.html to change texts, images, or add/remove plants.
3. Replace placeholder image URLs (placehold.co) with real image links (or local images).
4. To generate QR codes for the page, use any QR generator with the deployed URL, or create batch QR via Python script.

If you want, I can also:
- add a batch_qr.py script to generate QR images for multiple plot URLs,
- generate a zip with example CSV and QR images,
- or directly fill the template with your plant data and photos.
